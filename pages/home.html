Make a Request

We support POST requests for the follwoing endponts, each followed by your valid API key:

  • /add/
  • /subtract/
  • /multiply/
  • /divide/

The body of the request should be a JSON object with fields "a" and "b" containing the two numbers. For example, to add 3 to 4 you would POST the following body to /add/{your_key_here}:

{"a":3, "b":4}

Which would yield a response of:

{"ans":7}

If you make a mistake in your request you will likely get an error code 400 with a description of what you did wrong.
